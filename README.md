# BlackRoad Shader Library

GLSL shader collection and management system with code generation and validation.

## Features

- **Shader Types**: Vertex, fragment, compute shader support
- **Effect Generators**: Bloom, blur, vignette, chromatic aberration, scanlines, noise, warp, pixelate, edge-detect, color-grade
- **Validation**: Basic GLSL syntax checking (braces, semicolons, entry points)
- **Composition**: Blend multiple shaders with additive/multiply/screen/overlay modes
- **Export/Import**: JSON bundle serialization

## Usage

```bash
python shader_library.py
```
